Apa itu Machine Learning?

Machine Learning (ML) adalah sub-bidang Kecerdasan Buatan (AI) yang berfokus pada pengembangan algoritma yang memungkinkan komputer untuk belajar dari data dan meningkatkan performanya secara otomatis seiring bertambahnya informasi.

Pesan Dosen: Inti dari ML bukanlah pemrograman manual, melainkan pengenalan pola (pattern recognition).

Supervised vs Unsupervised

Supervised Learning

Pembelajaran yang diawasi menggunakan dataset yang sudah memiliki label. Tujuannya adalah memetakan input ke output berdasarkan contoh yang ada.

  • Klasifikasi: Target kategori (Spam/Bukan).
  • Regresi: Target numerik (Harga Rumah).

Unsupervised Learning

Pembelajaran tanpa pengawasan. Algoritma mencari struktur atau pola tersembunyi dari data yang tidak berlabel.

  • Clustering: Pengelompokan data serupa.
  • Dimensionality Reduction: Penyederhanaan fitur.

k-Nearest Neighbors (kNN)

Algoritma berbasis jarak yang mengklasifikasikan data baru berdasarkan label mayoritas dari 'k' tetangga terdekatnya.

Pseudocode / Logika

1. Tentukan jumlah tetangga (k)
2. Hitung jarak (Euclidean) antara data baru dengan semua data training
3. Urutkan jarak dari yang terkecil ke terbesar
4. Ambil k tetangga terdekat
5. Ambil mayoritas label dari k tetangga tersebut sebagai prediksi

Naive Bayes

Berdasarkan Teorema Bayes dengan asumsi "Naive" (Polos) bahwa setiap fitur bersifat independen. Sangat efektif untuk data teks.

P(A|B) = [P(B|A) * P(A)] / P(B)

Rumus di atas digunakan untuk menghitung probabilitas posterior kelas A jika diketahui bukti B.

Support Vector Machine (SVM)

Algoritma yang mencari Hyperplane (bidang pemisah) terbaik yang memaksimalkan Margin (jarak) antara kelas yang berbeda.

Evaluasi Model

Bagaimana mengukur kejeniusan model kita? Gunakan matriks evaluasi yang tepat!

Metrik Definisi Kapan Digunakan?
Accuracy Total prediksi benar / Total data. Data kelas seimbang.
Precision Ketepatan menebak positif. Menghindari False Alarm.
Recall Sensitivitas menemukan data positif. Kasus krusial (Penyakit/Bencana).

Pengantar Deep Learning

Deep Learning adalah evolusi dari Machine Learning yang menggunakan struktur berlapis yang disebut Neural Networks. Berbeda dengan ML klasik, Deep Learning melakukan ekstraksi fitur secara otomatis.

Input Layer: Menerima data mentah (piksel, sinyal).

Hidden Layers: Mengolah pola abstrak (garis, bentuk).

Output Layer: Hasil akhir (Klasifikasi/Prediksi).
